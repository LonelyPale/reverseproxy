# Reverse Proxy Debug Server


## 介绍

TCP 反向代理服务器,监听本地端口,并将请求全部转发到远端,
并将接收到的请求和远端的响应在标准输出中打印出来,方便做网络开发调试

可以用来做任何 TCP 连接的反向代理调试,抓包.
